# Travel Companion App

## Overview
The Travel Companion App is an Android application designed to assist international travellers in converting essential values such as currency, fuel-related units, and temperature.

This app provides a simple and user-friendly interface, allowing users to quickly perform conversions while travelling.

---

## Features

### 1. Currency Conversion
- Supported currencies: USD, AUD, EUR, JPY, GBP
- Uses fixed conversion rates provided in the assignment
- Converts between any two supported currencies

### 2. Fuel-related Conversions
Includes:
- Fuel efficiency (mpg ↔ km/L)
- Volume (Gallon ↔ Liter)
- Distance (Nautical Mile ↔ Kilometer)

Invalid conversion pairs are prevented.

### 3. Temperature Conversion
- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin

Supports negative values for temperature.

### 4. Input Validation
- Prevents empty input
- Handles invalid numbers
- Prevents negative values for currency and fuel
- Prevents invalid fuel unit combinations

### 5. Result Formatting
- Results are displayed with two decimal places for clarity

---

## User Interface

The application includes:
- A category selection dropdown
- Two unit selection dropdowns (From / To)
- A value input field
- A convert button
- A result display area

---

## Technologies Used
- Java
- Android Studio
- XML for UI design

---

## How to Use
1. Select a conversion category (Currency, Fuel, or Temperature)
2. Choose the "From" unit and "To" unit
3. Enter the value to be converted
4. Press the **Convert** button
5. View the result displayed below
